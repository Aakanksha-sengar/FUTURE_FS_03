## ✨ Features
- ✅ Fully responsive (mobile + desktop)
- ✅ Smooth scroll navigation
- ✅ Interactive menu with filter tabs (Cakes / Pastry / Snacks / Special)
- ✅ WhatsApp order integration on every menu item
- ✅ Contact form with direct WhatsApp redirect
- ✅ Scroll reveal animations
- ✅ Google Maps embed
- ✅ Floating WhatsApp button
- ✅ Marquee ticker strip
- ✅ Real business info (address, phone, timings)
